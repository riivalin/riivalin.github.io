成績評測
```text
>= 90 輸出 A
>= 80 輸出 B
>= 70 輸出 C
>= 60 輸出 D
< 60 輸出 E
```
## 使用 if-else if
```c#
Console.WriteLine("**請輸入成績***");
int score = Convert.ToInt32(Console.ReadLine());
if (score >= 90)
{
    Console.WriteLine("A");
} else if (score >= 80)
{
    Console.WriteLine("B");
} else if (score >= 70)
{
    Console.WriteLine("C");
} else if (score >= 60)
{
    Console.WriteLine("D");
} else
{
    Console.WriteLine("E");
}
```
## 使用 switch case
將範圍變成一個定值
```c#
Console.WriteLine("請輸入成績");
int score = Convert.ToInt32(Console.ReadLine()); //0-100

switch (score/10) //將範圍score 變成一個定值
{
    case 10: //case10, case9 執行的程式一樣
    case 9:
        Console.WriteLine("A");
        break;
    case 8:
        Console.WriteLine("B");
        break;
    case 7:
        Console.WriteLine("C");
        break;
    case 6:
        Console.WriteLine("D");
        break;
    default:
        Console.WriteLine("E");
        break;
}
```
