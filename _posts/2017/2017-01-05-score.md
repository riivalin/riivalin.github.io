成績評測
```text
>= 90 輸出 A
>= 80 輸出 B
>= 70 輸出 C
>= 60 輸出 D
< 60 輸出 E
```

```c#
Console.WriteLine("**請輸入成績***");
int score = Convert.ToInt32(Console.ReadLine());
if (score >= 90)
{
    Console.WriteLine("A");
} else if (score >= 80)
{
    Console.WriteLine("B");
} else if (score >= 70)
{
    Console.WriteLine("C");
} else if (score >= 60)
{
    Console.WriteLine("D");
} else
{
    Console.WriteLine("E");
}
```
