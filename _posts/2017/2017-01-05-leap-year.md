判斷閏年
- 年份能夠要400整除.(2000)
- 年份能夠被4整除，但不能被100整除. (2008)  

```c#
Console.WriteLine("**請輸入年份***");
int input = Convert.ToInt32(Console.ReadLine());

if ((input % 400 == 0) || (input % 4 == 0 && input % 100 != 0)) {
    Console.WriteLine($"{input}是閏年");
}
Console.ReadKey();
```
