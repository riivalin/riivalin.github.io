for死循環：`for(;;){...}`
```c#
for (; ; ) {
    Console.WriteLine("Dead Loop");
}
```
while死循環：`while(true){...}`
```c#
while (true) {
    Console.WriteLine("Dead Loop");
}
```
do while死循環：`do {...} while(true);`
```c#
do {
    Console.WriteLine("Dead Loop");
} while (true);
```
