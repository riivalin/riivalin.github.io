## 輸入
```c#
Console.Read(); // 讀入一個字元
Console.ReadLine(); // 讀入一行字串
```

## 舉例
輸入字元
```c#
char c = Convert.ToChar(Console.Read());
```

輸入字串
```c#
string s = Console.ReadLine();
```
輸入整數
```c#
int n = Convert.ToInt32(Console.ReadLine());
```

## 輸出
使用大括弧`{0}` `{1}` 佔符位，後面放參數
```c#
Console.WriteLine("{0} and {1}", x, y); 
```
使用`$”{參數}”`
```c#
Console.WriteLine($"{x} and {y}"); 
```
