## 單引號「`'`」
一般表示一個字串

```sql
update customer set name = 'Rii'
```

## 雙引號
一般表示一個變量，如：資料表名、欄位名、別名等

```sql
update "customer" set "name" = 'Rii'
select name as "姓名" from customer
```
