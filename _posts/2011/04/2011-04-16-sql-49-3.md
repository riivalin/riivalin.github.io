## 語法

```sql
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```

## 範例

將數位相機(id=6)的進貨價格更新為50000、販賣價格更新為70000

```sql
--將數位相機(id=6)的進貨價格更新為50000、販賣價格更新為70000
update 商品清單
set 進貨單價=50000, 販賣單價=70000
where 商品ID=6
```

執行結果：
```
select * from 商品清單 where 商品ID=6

商品ID	商品名稱	群組名稱	進貨單價	販賣單價
6	數位相機	周邊設備	50000	70000
```
