```sql
--將產品名稱內所有的英文轉換成小寫
select lower(商品名稱) from 商品清單
```

## Lower() 轉小寫

```sql
select lower('ABCD') --abcd
```

## Upper() 轉大寫

```sql
select upper('abcd') --ABCD
```
