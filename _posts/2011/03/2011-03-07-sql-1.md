- Developer 開發版(use it)
- Enterprise 企業版
- Express 試用版

免費版本有 Developer (開發版) 及 Express。

- Developer (開發版): 功能完整開放使用，但只限制於開發環境使用，不可應用於生產環境。
- Express: 限制空間、記憶體、CPU 大小，關閉 SQL Agent 服務，但可應用於生產環境使用。
