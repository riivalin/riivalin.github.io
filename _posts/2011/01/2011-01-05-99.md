巢狀迴圈
: 當遇到某個事情要做一遍，而另一個事情要做N遍的時候。

```c#
for (int i = 1; i < 10; i++)
{
    for (int j = 1; j < 10; j++)
    {
        Console.Write($"{i}*{j}={i * j}\t"); // \t tab鍵空格
    }
    Console.WriteLine(); //換行
}
Console.ReadKey();
```
