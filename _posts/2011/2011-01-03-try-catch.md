異常捕捉 try-catch

語法上沒有錯誤，在程式執行的過程當中，由於某些原因，程式出現了錯誤，不能再正常運行。
```text
try {
    可能會出現異常的程式碼
} catch {
    出現異常後要執行的程式碼
}
```
```c#
Console.WriteLine("請輸入一個數字:");
int num = 0;
bool b = true;
try
{
    //可能會出現異常的程式碼
    num = Convert.ToInt32(Console.ReadLine()); //賦值
} 
catch (Exception)
{
    //出現異常後要執行的程式碼
    b = false;
    Console.WriteLine("輸入的內容不能轉換成數字");
}

if (b) {
    Console.WriteLine(num * 2);
}
Console.ReadKey();
```
