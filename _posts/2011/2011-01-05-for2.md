找出100-999水仙花數

當前這個百位數字 =百位的立方+十位的立方+個位的立方
153 =  1³ + 5³ + 3³

百位：153/100
十位：153%100/10
個位：153%10

```c#
for (int i = 100; i <= 999; i++)
{
    int bai = i / 100;
    int shi = i % 100 / 10;
    int ge = i % 10;
    if (i == bai * bai * bai + shi * shi * shi + ge * ge * ge) {
        Console.WriteLine($"{i} 是水仙花數");
    }
}
```
